Features

主な機能

🎙

高音質録音

16kHz / Mono / 16bit PCMでクリアに録音。一時停止・再開にも対応。Foreground Serviceによるバックグラウンド録音で、アプリを閉じても録り続けられる。

🤖

AI文字起こし & 要約

Groq Whisperによる高速文字起こし。Geminiが会議内容を構造化して要約。タイトル・概要・トピック・決定事項・アクションアイテムを自動生成。要約指示のカスタマイズも可能。

How It Works

3ステップで完了

難しい設定は一切不要。録音を開始するだけで、あとはAIにお任せ。

1

Record

ボタンひとつで録音開始。バックグラウンドでも動作し、長時間の会議も安心。

2

Transcribe

Groq Whisperが高精度で文字起こし。ローカルWhisperも選択可能。

3

Summarize

GeminiまたはローカルLLMが会議内容を整理。見やすい要約を自動生成。

Cross Platform

Android & macOS

スマホでもデスクトップでも。使いたい環境で使える。

📱 Android

  • Foreground Serviceによるバックグラウンド録音
  • セッション履歴・要約一覧管理
  • 録音時間・空き容量リアルタイム表示
  • 録音一時停止・再開対応
  • 要約指示のカスタマイズ
  • 音量レベルインジケーター

💻 macOS

  • Metal GPU 加速（Apple Silicon）
  • ローカル LLM (llama.cpp) 対応
  • ローカル Whisper 対応
  • Homebrew / DMG インストール
  • クラウド API 併用も可能（Groq / Gemini）

🔒 プライバシー＆料金

ローカル処理で安心

文字起こし・要約は全てローカルでも実行可能。音声データが外部に送信されることはありません。

クラウドAPI（Groq / Gemini）を利用する場合も、各サービスに無料枠あり。APIキーは端末内で暗号化管理されます。
